eyeareseestats
##############

A tool for collecting stats from IRC logs

Metrics monitored
------------------

- topic
- joins/leaves
- bans/kicks
- actions
- chats
- mentions (source and dest)
- nick changes
- op/deop
- links

Outputs
-------

- rst, so that you can easily use :code:`rst2html` etc., to generate a fancy html page.
